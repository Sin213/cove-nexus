### Bug Fixes

- **Detect and surface failed launches:** Previously, if a program failed to start (missing DLL, antivirus block, crash on startup), Nexus would silently show "launched" while nothing appeared in Task Manager. Now Nexus captures stderr from the child process for the first 5 seconds and probes the PID at 600 ms and 2.5 s to verify the process is actually alive. If it isn't, a "Launch failed" error is shown with the real reason instead of a false success.

- **Show crash toast for immediate exits:** When a launched program exits with a non-zero code within 5 seconds of starting, Nexus now shows a toast notification with the program name, exit code, and any captured stderr output.
