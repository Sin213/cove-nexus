### Bug Fixes

- **Fix file opening in child apps on KDE:** Tools launched from Nexus (e.g. Cove File Toolkit) could not open files via right-click → Open. The root cause was that `KDE_SESSION_VERSION` and other KDE session variables were not forwarded to child processes, causing `xdg-open` to fall back to `kfmclient` (which doesn't exist on KDE 6) instead of `kde-open`. Added `KDE_` to the environment prefix allowlist so all KDE session vars are passed through.

- **Strip Electron-polluted linker vars from child launch env:** Removed `LD_LIBRARY_PATH`, `LD_PRELOAD`, and `GDK_PIXBUF` vars from the curated launch environment on Linux. Electron and AppImage runtimes inject their own bundled-lib paths into these variables; forwarding them to child AppImages broke their own `xdg-open` calls by polluting the linker search path.
