## Cove Nexus v2.0.8

UI font scaling — zoom the entire interface with a keyboard shortcut.

### Added

- **Ctrl/Cmd + `+` / `-` / `0` zooms the app.** `+` increases the global font size, `-` decreases it, `0` resets to default. Range is 13–22px in 1px steps. The choice persists across restarts via the existing tweaks config, so reopening the app keeps your preferred size.

### Changed

- **Stylesheet refactor: `px` → `rem`.** Every hardcoded `px` font-size in `renderer/index.html` was converted to `rem` so the whole UI scales coherently from a single root anchor. `html, body` is now pinned at `font-size: 1rem`, and the zoom shortcut shifts that root size — chips, cards, modals, tooltips, banners, segmented controls, and the tweaks panel all track together. No layout changes; purely a unit swap so text scales as one system.
