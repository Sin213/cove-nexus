Refreshes the app cards to show the current release version for every registry app and prefers per-artifact .sha256 sidecars when verifying downloads.

### Bug Fixes

- **Stale card versions:** The fallback version and date shown on not-installed cards were out of date for 9 of the 10 registry apps. All entries now match the latest GitHub releases.

- **Download verification order:** Checksum verification now looks for the per-artifact `<name>.sha256` sidecar first and only falls back to a legacy bundled checksum file on older releases.
