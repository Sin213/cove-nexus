### Bug Fixes

- **Reconcile installed apps from on-disk artifact versions instead of trusting installs.json:** Apps that update themselves (electron-updater renaming the AppImage, the Python fleet installing under a new versioned filename) left `installs.json` pointing at dead paths or stale tags. This caused phantom update badges, false not-installed states, and the startup cleanup sweep deleting binaries that a tool had just self-updated to. Nexus now repoints the registered tag and path to match what is actually on disk before reading the registry, and the cleanup sweep never deletes a file newer than the registered version. This release is backward compatible with apps that have not yet picked up the new rename-on-update behavior.

Thanks to @kraibse for pointing this out.
