### Bug Fixes

- **Fix launch regression on paths with spaces:** Programs installed on paths containing spaces (e.g. `F:\vimms and stuff\...`) failed to launch with "'F:\vimms' is not recognized as an internal or external command". The `shell: true` spawn added in v2.2.5 passed the exe path unquoted through `cmd.exe /s`, which stripped the outer quotes and split on spaces. The path is now inner-quoted so it survives cmd.exe parsing. Regression introduced in v2.2.5.
