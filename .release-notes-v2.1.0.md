## Cove Nexus v2.1.0

**Foxy Mode** — supported Cove tools now open as a tab inside Nexus instead of a separate window. Cove Meme Maker is the first tool to ship with it polished and ready.

### Added

- **Foxy Mode.** Tools that advertise an embedded web UI can now open directly inside Nexus. The card swaps **Launch** for **Running** while the tab is open, and clicking **Running** focuses the existing tab instead of opening a duplicate. Closing the tab returns the card to its normal **Launch** state.
- **Cove Meme Maker in Foxy Mode.** Meme Maker is wired up as the first polished Foxy Mode app. Modern-mode live preview works inside the embedded tab. For the best experience, update Cove Meme Maker to **v2.1.1** (or newer).

### Changed

- **"Running" replaces "Focus App".** When a Foxy Mode app is already open, its card and primary action now read **Running**, and clicking it snaps focus back to the existing tab — Nexus no longer opens duplicates.
- **Launcher chrome is hidden for embedded apps.** Foxy Mode tabs render without the launcher's sidebar/titlebar so the embedded app gets the full window.

### Removed

- **Dev-only "Nexus tab-web smoke test" card.** The internal smoke harness card no longer appears in the normal app library.

### Preserved

- Standard install / launch / update behavior for every regular tool — Foxy Mode is opt-in per tool, and tools without it continue to launch as standalone apps exactly as before.
- GitHub release discovery, version pinning, installed/adopted program handling, and silent self-update for Setup.exe + AppImage.
- The security sandbox around embedded views — no sandbox weakening for Foxy Mode.

### Auto-updates

Setup.exe + AppImage users get this silently on next launch. Portable users will see the in-app banner.

### Heads up

Update **Cove Meme Maker** to **v2.1.1** or newer to get the polished Foxy Mode experience inside Nexus.
