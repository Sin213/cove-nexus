### Bug Fixes

- **Missing lib/ directory in packaged build:** The reconcile module was not included in the electron-builder files list, causing Nexus to crash on launch with "Cannot find module './lib/reconcile'". Added "lib/**/*" to the build files so the module is bundled correctly across all platforms (Windows portable/setup, AppImage, deb).

- **Per-artifact SHA256 sidecars:** Each shipped binary and auto-update metadata file now gets its own `<artifact>.sha256` sidecar file on the release.
